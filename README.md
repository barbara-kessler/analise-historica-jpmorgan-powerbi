# analise-historica-jpmorgan-powerbi
Dashboard de análise de performance de ativos da JPMorgan (1980–2026). Apresenta indicadores de retorno aual, valor acumulado, crescimento YTD, volatilidade e volume negociado. Permite avaliar liquidez, risco e tendências de mercado. ⚠️ Dados de 2026 disponíveis apenas até abril.
